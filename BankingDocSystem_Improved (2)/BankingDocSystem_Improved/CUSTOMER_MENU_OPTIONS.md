## Menu Options for Customers

### 1. **🏠 Dashboard** (Default View)
**What it shows:**
- Welcome message
- Personal information overview
- All bank accounts with balances
- Quick statistics
- Account summaries

**How to access:**
- Click "🏠 Dashboard" in sidebar
- Opens automatically when login

---

### 2. **💸 Transactions**
**What it shows:**
- Complete transaction history
- Transactions per account
- Transaction types (Deposit, Withdrawal, Transfer)
- Amounts and dates
- Filter by account

**How to access:**
- Click "💸 Transactions" in sidebar
- Select account from dropdown
- See all transactions

---

### 3. **📄 Documents** ⭐ CUSTOMER VERSION
**What it shows:**
- All uploaded documents
- Document type (Passport, ID, etc.)
- File name and size
- Upload date
- Approval status (color-coded)
- Review notes (if any)

**How to access:**
- Click "📄 Documents" in sidebar
- Search by document type or filename
- Click "👁️ View" to preview image
- Click "⬇️ Download" to save file

**Features:**
- Preview document images
- Download documents to computer
- Search functionality
- Status indicators

---

### 4. **👤 Edit Profile** ⭐ NEW!
**What it shows:**
- Profile photo (circular display)
- Personal information form
- Login information (read-only)
- Password change section

**What you can do:**
```
📸 Upload/Change Photo:
   Click "🖼️ Upload Photo"
   Select JPG or PNG (max 5MB)
   Photo displays in circle
   Click "Save All Changes"

✏️ Edit Personal Info:
   Update Full Name
   Update Phone (7-15 digits)
   Update Address
   Update Date of Birth
   Cannot change Username or Email
   Click "Save All Changes"

🔒 Change Password:
   Enter current password
   Enter new password (6+ chars)
   Confirm password
   Click "Save All Changes"
```

**How to access:**
- Click "👤 Edit Profile" in sidebar (under "MY ACCOUNT")
- See modern, attractive interface
- Edit any field
- Upload new photo
- Save all changes

## Step-by-Step Workflows

### To View Documents
```
1. Click "📄 Documents" in sidebar
   ↓
2. See list of all your documents
   ↓
3. Use search box to find documents
   ↓
4. Click "👁️ View" to see document
   ↓
5. Document viewer opens with preview
   ↓
6. Click "⬇️ Download" to save to computer
   ↓
7. Click "❌ Close" to exit viewer
```

### To Edit Profile
```
1. Click "👤 Edit Profile" in sidebar
   ↓
2. See profile page with sections:
   - Profile photo (top)
   - Personal information (left)
   - Password change (right)
   ↓
3. Upload photo:
   - Click "🖼️ Upload Photo"
   - Choose JPG/PNG file
   - Photo shows in circle
   ↓
4. Edit personal information:
   - Change Full Name
   - Change Phone
   - Change Address
   - Change Date of Birth
   ↓
5. Change password (optional):
   - Enter current password
   - Enter new password
   - Confirm password
   ↓
6. Click "💾 Save All Changes" at bottom
   ↓
7. See success message
   ↓
8. Profile is updated
```

### To View Transactions
```
1. Click "💸 Transactions" in sidebar
   ↓
2. See all transactions
   ↓
3. Select account from dropdown
   ↓
4. See transactions for that account
   ↓
5. View type, amount, date
```

### To View Dashboard
```
1. Click "🏠 Dashboard" in sidebar
   ↓
2. See overview:
   - Your personal info
   - All your accounts
   - Account balances
   - Quick statistics
```

## Important Notes

### Profile Changes
- Click "Save All Changes" to save
- Changes happen immediately
- Click "Cancel" to discard changes

### Password Security
- Current password must be correct
- New password must be 6+ characters
- Must match confirmation
- Never saved in plain text

### Photo Upload
- Must be JPG or PNG
- Maximum size: 5MB
- Saved with timestamp
- Can be removed anytime

### Documents
- Only see your documents
- Click View to preview
- Click Download to save
- Cannot delete documents

## Troubleshooting

### "I don't see Edit Profile option"
**Solution**: You must be logged in as a customer. Check menu on left sidebar under "MY ACCOUNT" section.

### "Edit Profile button isn't working"
**Solution**: Refresh the page or logout and log back in.

### "Can't upload photo"
**Solution**: Check that file is JPG or PNG, and file size is less than 5MB.

### "Can't see my documents"
**Solution**: Go to Dashboard first to ensure profile loads, then click Documents.

### "Documents list is empty"
**Solution**: You may not have uploaded any documents yet. Contact admin to upload documents.
